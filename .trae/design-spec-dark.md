## 三、暗色效果调整详情

### 3.1 麦克风按钮投影

暗色主题下调整了投影参数以保持悬浮感：

| 投影层 | 亮色值 | 暗色值 | 变化说明 |
|--------|--------|--------|---------|
| **红晕散** | `rgba(220,38,38,0.15)` offset:y=8 r=30 | `rgba(239,68,68,0.20)` offset:y=8 r=30 | 透明度 0.15→0.20 |
| **近投影** | `rgba(220,38,38,0.30)` offset:y=6 r=24 | `rgba(239,68,68,0.40)` offset:y=6 r=24 | 透明度 0.30→0.40 |
| **远投影** | `rgba(220,38,38,0.10)` offset:y=16 r=48 | `rgba(239,68,68,0.15)` offset:y=16 r=48 | 透明度 0.10→0.15 |

### 3.2 波形动画环透明度

暗色背景下增强透明度以确保可见性：

| 层 | 亮色透明度 | 暗色透明度 |
|----|-----------|-----------|
| **Ring 1（外层）** | 15% | 20% |
| **Ring 2（中层）** | 25% | 35% |
| **Ring 3（内层）** | 50% | 60% |

### 3.3 卡片投影

| 属性 | 亮色值 | 暗色值 |
|------|--------|--------|
| **投影色** | `rgba(0,0,0,0.06)` | `rgba(0,0,0,0.25)` |
| **偏移** | y:4 | y:4 |
| **模糊** | 20px | 20px |

## 七、亮暗切换实现建议

### 7.1 CSS 变量方案

```css
:root {
  /* 亮色主题 (默认) */
  --bg-primary: #FAFAFC;
  --bg-surface: #FFFFFF;
  --bg-card: #FFFFFF;
  --text-primary: #0F172A;
  --text-secondary: #64748B;
  --text-tertiary: #94A3B8;
  --border-default: #FAE3E3;
  --btn-surface: #FDF2F2;
  --color-record: #DC2626;
  --color-wave: #2563EB;
  --shadow-card: rgba(0, 0, 0, 0.06);
}

[data-theme="dark"] {
  --bg-primary: #0B1120;
  --bg-surface: #111827;
  --bg-card: #1A2332;
  --text-primary: #F1F5F9;
  --text-secondary: #94A3B8;
  --text-tertiary: #64748B;
  --border-default: #2D3643;
  --btn-surface: #1E293B;
  --color-record: #EF4444;
  --color-wave: #3B82F6;
  --shadow-card: rgba(0, 0, 0, 0.25);
}
```

### 7.2 prefers-color-scheme 自动适配

```css
@media (prefers-color-scheme: dark) {
  :root:not([data-theme="light"]) {
    /* 暗色主题变量 */
  }
}
```

## 九、实现备注

1. **无色相偏移**：暗色主题仅调整明度/饱和度，保持品牌色（红+蓝）的核心识别不变
2. **对比度合规**：所有文字-背景组合通过 WCAG AA 标准（对比度 ≥ 4.5:1）
3. **OLED 友好**：背景使用 `#0B1120` 而非纯黑 `#000000`，避免纯黑带来的拖影问题，同时保持低功耗
4. **投影增强**：暗色下投影需增加透明度才能保持可见性，已按比例调整
5. **布局稳定**：所有间距、圆角、字号与亮色主题完全一致，切换时无布局重排
